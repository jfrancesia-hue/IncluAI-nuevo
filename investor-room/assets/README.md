## Especificaciones técnicas

- **Logos:** SVG vectorial + PNG 2x en varios tamaños (256, 512, 1024 px).
- **Screenshots:** 1920×1080 mínimo, formato PNG sin compresión, con bordes redondeados si es producto mobile.
- **Video:** 1920×1080, 30fps, mp4 H.264, audio AAC 48kHz. Máx 100 MB para video de 5 min.
- **Fotos profesionales:** mínimo 3000 px lado largo, JPG 90% calidad.
- **Gráficos:** preferir SVG para poder editar; si es PNG, 2x.

---

## Herramientas recomendadas

- Diseño de deck: **Pitch.com** o **Figma + plugin Slides**.
- Mockups de pantalla: **Figma + plugin Mockup** o **Shots.so**.
- Screenshots con marco: **Xnapper** (Mac), **Shottr**, **CleanShot X**.
- Video demo: **Loom** para captura rápida, **Descript** para edición con sincronización.
- Gráficos financieros: **Flourish** o **Datawrapper** (no Excel).
- Diagramas técnicos: **Excalidraw** (para mocks) o **Lucidchart** (para arquitectura formal).
